Интерфейс & Программатор роботов

Система UI Toolkit, автоматическое связывание данных и графический редактор алгоритмов.

Интерфейс на базе UI Toolkit

UI клиента Fodinae построен на современном фреймворке Unity UI Toolkit (вместо устаревшего uGUI). Компоновка интерфейса определяется XML-документами UXML, а стилизуется с помощью USS-таблиц.

Кастомные контролы:

  • Selectable — базовый абстрактный класс для элементов пользовательского интерфейса, поддерживающих выбор курсором (селекторы хотбара, ячейки инвентаря).
  • RegexTextField — текстовое поле ввода с валидацией вводимых символов в реальном времени с помощью регулярных выражений (применяется для ввода никнеймов, числовых IP-адресов, портов).
  • ChatInput — менеджер фокуса клавиатурного ввода чата. Блокирует прохождение событий перемещения персонажа (PlayerMovementController), когда пользователь вводит сообщение в текстовое поле чата.
  • ClickContextResolver — парсит и разрешает клики по вложенным элементам VisualElement, сопоставляя их с контекстными действиями.

Связывание данных (WindowBinding)

Для автоматического отображения игровых параметров (здоровье, энергия, описание предметов) используется скрипт связывания данных WindowBinding на базе шаблонизатора SmartFormat.

Принцип работы:

  1. Компонент сканирует дерево элементов VisualElement окна.
  2. Находит элементы Label, содержащие в тексте шаблоны вида {Player.Hp} / {Player.MaxHp}.
  3. Автоматически подписывается на изменения соответствующих моделей данных (PlayerStatsModel, InventoryModel).
  4. Для выполнения математических операций внутри шаблонов подключен парсер LogiCalcFormatter.
  5. При любом изменении данных текст элементов обновляется автоматически без написания явного кода перерисовки.

Визуальный программатор роботов

Система программатора позволяет игроку визуально составлять алгоритмы поведения для роботов на специальном графическом поле.

Составные модули программатора:

  • ProgrammatorGrid:
    • Графическая сетка, отображающая блоки команд и связи между ними.
    • Обрабатывает перетаскивание блоков (drag and drop), соединение входов и выходов логических узлов.
    • Рисует визуальные линии связей с помощью геометрических примитивов.
  • ProgrammatorData — структура данных, содержащая полное сериализованное представление алгоритма. Она сохраняется в JSON и отправляется на сервер для выполнения роботом.
  • RadialMenu — радиальное круговое меню быстрого выбора команд. При клике на пустую ячейку сетки программатора оно отображает круговое меню доступных команд (движение, копка, использование, условия), позволяя быстро добавить нужную команду.
